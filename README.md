# Модель классификации пользователей сервиса доставки кофе склонных к оттоку 

## Описание

Модель LogisticRegressor классифицирует пользователей склонных к отказу от использования сервиса 

## Требования

- **Python**: 3.11 или выше
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

### Шаг 1: Установка зависимостей

Создайте файл `.env` на основе шаблона `.env.example`:

```bash
cp .env.example .env
```

Затем отредактируйте `.env` и укажите ваши данные для подключения к PostgreSQL.

### Шаг 2: Запуск скрипта установки

#### Для Fish shell:

```fish
fish start.fish
```

#### Для Bash/Zsh:

```bash
./start.sh
```

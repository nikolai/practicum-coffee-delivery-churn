# Модель классификации пользователей сервиса доставки кофе склонных к оттоку

## Описание

Модель LogisticRegressor классифицирует пользователей склонных к отказу от использования сервиса.

В работе представлен пример EDA, построения пайплайна обучения, создание кастомного FunctionTransformer для преобразования пропусков и аномальных значений.

Благодаря GridSearchCV производится поиск оптимальных параметров модели с помощью кросс-валидации.

Также произведен feature engineering, который включает в себя, который в том числе поднял качество модели.

Выводы содержат рекомендации предпринимаемых шагов для улучшения состояния оттока пользователей.

## Требования

- **Python**: 3.11 или выше
- **ОС**: Linux, macOS или Windows (с WSL)

## Установка и запуск

#### Для Fish shell:

```fish
fish start.fish
```

#### Для Bash/Zsh:

```bash
./start.sh
```
